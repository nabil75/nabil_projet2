```python
- Créer un répertoire en local et positionnez-vous sur le répertoire créé

- cloner le repo en lançant la commande : $ git clone https://github.com/nabil75/nabil_projet2.git
- Une fois le repo cloné, vous pourrez lancer chacune des étapes du projet en vous positionnant en ligne de commande dans chacun des sous-répertoires correspondant (ex all pour extraire tous les livres du site)
- Exemple : pour lancer l'étape1 (récupérer les caractérisques d'un livre) positionnez-vous sur le répertoire ** path/to/your/folder/ ** puis lancer le script en invoquant la commande suivante : python get_one_book.py
```
